# Ohjeita
## lataaminen
```
git clone https://github.com/nicohi/otm-harjoitustyo
cd otm-harjoitustyo
```
## suorittaminen
```
mvn compile exec:java -Dexec.mainClass=nicohi.planetsim.Main
```
## testaus
```
mvn test
```
## automaattinen raportti
```
mvn test jacoco:report
```
